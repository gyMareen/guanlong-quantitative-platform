# Longport Platform Frontend

基于 Vue 3 + Vite + TypeScript + Element Plus 的前端项目。

## 技术栈

- Vue 3.4
- Vite 5
- TypeScript 5
- Vue Router 4
- Pinia 2
- Element Plus 2.5
- Axios
- SCSS

## 环境要求

- Node.js 18+
- pnpm / npm / yarn

## 快速开始

### 1. 安装依赖

```bash
npm install
# 或
pnpm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问: http://localhost:5173

### 3. 构建生产版本

```bash
npm run build
```

## 项目结构

```
src/
├── api/           # API 接口
├── assets/        # 静态资源
├── components/    # 公共组件
├── router/        # 路由配置
├── store/         # 状态管理 (Pinia)
├── styles/        # 全局样式
├── utils/         # 工具函数
├── views/         # 页面组件
├── App.vue        # 根组件
└── main.ts        # 入口文件
```
